# 用語・言語統一ルール

## 言語方針
- タイトル: 英語（既存パターン名は変更しない）
- セクション見出し: 日本語
- 説明文: 日本語
- コード: 英語（コメントも英語）
- 記号: ✅/❌は許容

## 統一用語
- build automation → ビルド自動化
- deployment → デプロイメント
- rollback → ロールバック
- monitoring → 監視
- testing → テスト
- pipeline → パイプライン

## フォーマット
```markdown
# English Title (既存のパターン名)

## 日本語セクション

### 日本語サブセクション
- 日本語説明文
```
